Product Name:
Eyes First – Double Up

Description:
Eyes First – Double Up is a puzzle with numbered tiles, where the goal is to slide the tiles around until the number 2048 is reached. Tiles will only combine if they are the same number (2 or 4 initially, and powers of two thereafter). Try to create a tile with the number 2048. One can still continue playing after the number 2048 is reached.
This app is also designed to be played with eye control using a Windows 10 compatible eye tracker.
